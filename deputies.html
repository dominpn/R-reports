Pobieranie danych z Internetu - modyfikacja skryptu

W skrypcie dokonano następujących zmian:

  • dodano wyciąganie danych związanych z okręgiem wyborczym
pola <- c("Liczba głosów:", "Klub/koło:", "Data i miejsce urodzenia:", "Wykształcenie:", "Zawód:", "Nazwisko:", "Okręg wyborczy:")

colnames(df) <- c("Liczba.Glosow", "Klub", "Data", "Wyksztalcenie", "Zawod", "ImieNazwisko", "OkregWyborczy")
  • poprawiono wyciąganie wieku na podstawie datu urodzenia; w tym celu użyto funkcji age_calc, następnie uzyskany rezultat zaokrąglono w dół za pomocą funkcji floor
poslowie$Wiek <- floor(age_calc(poslowie$DataUrodzenia, units = "years"))
  • dodano wyciąganie danych na temat płci na podstawie imienia; w tym celu stworzono funcję get_gender, która dzieli ciąg znaków według spacji ( strsplit), następnie z 1 elementu tablicy brany jest ostatni znak ( stri_sub), jeżeli jest to literka “a” jest to kobieta w przeciwnym razie mężczyzna
get_gender <- function(name) {
  if (all(stri_sub(unlist(strsplit(toString(name), " "))[1], -1, -1) == "a")) {
    return("K")
  } else {
    return("M")
  }
}

poslowie$Plec <- sapply(poslowie$ImieNazwisko,get_gender)
  • uzyskany wynik zostaje zapisany do pliku CSV w celu zapobiegnięcia ciągłemu pobierania danych ze strony (ciągłe odpytywanie prowadzi do tymczasowego zablokowanie adresu - kod błędu 403 Forbidden)
write.csv(poslowie,'deputies.csv', row.names = TRUE)

Efekt końcowy

poslowie <- list()
for (posel in 1:460) {
  strona <- read_html(paste0("http://sejm.gov.pl/sejm8.nsf/posel.xsp?id=",posel%/%100, (posel%/%10)%%10, posel %% 10))
  nazwisko <- html_text(html_nodes(strona, "h1"))
  
  pola <- html_text(html_nodes(strona, ".data p"))
  if (any(pola == "Wygaśnięcie mandatu:")) {
    ind <- which(pola == "Wygaśnięcie mandatu:")
    pola <- pola[-ind+c(0,-1)]
    if (!grepl(pola[ind], pattern=":$"))
      pola <- pola[-ind]
  }
  wartosci <- pola[seq(2, length(pola), 2)]
  names(wartosci) <- pola[seq(1, length(pola), 2)]
  wartosci[["Nazwisko:"]] <- nazwisko
  id <- as.character(posel)
  poslowie[[id]] <- wartosci
}

pola <- c("Liczba głosów:", "Klub/koło:", "Data i miejsce urodzenia:", "Wykształcenie:", "Zawód:", "Nazwisko:", "Okręg wyborczy:")

dane <- list()
for(pole in pola) {
  dane[[pole]] <- sapply(poslowie, function(posel) posel[pole])
}

df <- as.data.frame(dane)
colnames(df) <- c("Liczba.Glosow", "Klub", "Data", "Wyksztalcenie", "Zawod", "ImieNazwisko", "OkregWyborczy")
rownames(df) <- names(poslowie)
df$Klub <- gsub(df$Klub, pattern = "Klub Parlamentarny |Klub Poselski ", replacement="")
df$DataUrodzenia <- gsub(df$Data, pattern = ",.*$", replacement = "")
df$MiejsceUrodzenia <- gsub(df$Data, pattern = "^.*,.", replacement = "")

get_gender <- function(name) {
  if (all(stri_sub(unlist(strsplit(toString(name), " "))[1], -1, -1) == "a")) {
    return("K")
  } else {
    return("M")
  }
}

poslowie <- df[,-3]
poslowie$Liczba.Glosow <- as.numeric(as.character(poslowie$Liczba.Glosow))
poslowie$Klub <- factor(poslowie$Klub)
poslowie$DataUrodzenia <- dmy(poslowie$DataUrodzenia)
poslowie$MiejsceUrodzenia <- factor(poslowie$MiejsceUrodzenia)
poslowie$Wiek <- floor(age_calc(poslowie$DataUrodzenia, units = "years"))
poslowie$Plec <- sapply(poslowie$ImieNazwisko,get_gender)
write.csv(poslowie,'deputies.csv', row.names = TRUE)

Opis funkcji

  • list() - utworzenie listy

  • read_html(file) - odczytanie zawartości pliku .html

    • file - ścieżka do pliku .html

  • paste0() - konkatenacja serii ciągów znaków bez żadnego separatora

  • html_nodes(document, node) - wyciągncięcie z dokumentu podanego węzła

    • document - zawartość .html
    • node - jaki węzeł jest poszukiwany

  • html_text() - wyciągniecie wartości danego tagu, w przypadku skryptu powyżej szukamy klasy data oraz wyciągamy wszystkie wartości znajdujące się w znaczniku

  • any() - sprawdzane jest czy jakiś warunek podany w funkcji równa się TRUE

  • which() - zwracana są indeksy wektora, które spełniają warunek

  • length() - zwracana jest długość zmiennej

  • grepl() - zwraca prawdę jeżeli ciąg znaków zawiera podany wzorzec

  • sapply(x, fun) - zastosowanie podanej funkcji (fun) dla listy, wektora lub ramki danych (x)

  • c() - łączy podane argumenty w wektor

  • gsub() - zamiana znaków spełniających wzorzec

  • toString() - konwersja danej na typ string

  • strsplit() - podział łańcucha znaków według separatora

  • unlist() - zamiana listy na wektor

  • stri_sub() - pozwala na wydzielenie podciągu z ciągu znaków

  • all() - zwraca TRUE, jeżeli wszystkie warunki logiczne są prawdziwe

  • as.numeric() - konwersja na typ numeryczny

  • as.character() - konwersja na typ znakowy

  • as.data.frame() - konwersja na ramkę danych

  • factor() - zmiana wektora na czynniki

  • dmy() - parsowanie daty na dzień, miesiąc, rok

  • floor() - zaokrąglenie w dół

  • age_calc() - wylicza czas pomiędzy dwiema datami

  • write.csv() - zapis ramki danych do pliku csv
